Installation
============
1. Go to home directory
2. Git clone
	* For SSH:
		git clone git@github.com:jonzjia/dotfiles.git dotfiles

	* For HTTPS:
		git clone https://github.com/jonzjia/dotfiles.git dotfiles
3. Go into new dotfiles directory.
4. Run the shell script:
	./make.sh
5. Success
